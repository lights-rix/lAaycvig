# 项目简介

## 核心功能
- 眼科患者信息管理
- 患者复诊提醒
- 患者复诊随访数据记录
- 电子档案管理
- 用药记录管理
- 眼科疾病信息管理
- 系统管理
- 轮播图管理
- 个人中心

## 适用场景
- 眼科医疗机构
- 眼科患者管理
- 眼科疾病数据统计与分析

## 技术特点
- 基于 SpringBoot 和 Vue 框架，实现前后端分离
- 使用 Java 语言进行开发
- 支持MySQL数据库存储
- 采用 B/S 架构，支持多用户访问
- 提供数据可视化统计功能，使用 echarts 库
- 集成百度千帆智能AI技术
- 提供详细的万字文档和视频教程
